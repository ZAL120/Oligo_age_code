# Code Availability

This repository is associated with the manuscript:

**Donor Age and Oligosaccharide Structure Jointly Shape the Gut Microbiome Function**

The analysis code for this study is currently being organized for public release.

The repository will include scripts used for data processing, statistical analysis, response modeling, and figure generation. Additional documentation and required processed input files will be added as the repository is finalized.

Raw mass spectrometry proteomics data are available through PRIDE under the dataset identifier **PXD079344**.

A complete code release will be provided before publication.
